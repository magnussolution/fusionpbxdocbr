**FusionPBX**
---------------

Um projeto open source que prove uma interface web flexissivel e customizavel, realmente potente, escalavel e mult-plataforma para o voice swith chamado `FreeSWITCH`_.

FusionPBX pode ser instalado em varios sistemas operacionais e hardwares.(otimizado para Debian 8). 
FusionPBX tem uma interface com ilimitadas extensōes,voicemail-to-email, music on hold, call parking, linhas analogicas or cirquitos T1/E1 a muitas outras funçōes.
FusionPBX tem tudo o que pequenas, media e grandes empresas precisam para colocar em funcionamento um robusto sistema de PBX IP `Clique aqui para FusionPBX youtube Somente Ingles`_ .

Beneficios do FusionPBX
---------------------

1. Algumas funcionalidade extras para dar uma incrível robustez ao FreeSWITCH.
2. Torna o FreeSWITCH mais facil de administrar, e permite você ainda trabalhar com o FreeSWITCH Command Line Interface (fs_cli). 
3. Permite um dar para seus cliente um atrativo painel WEB para interatuar com voce.
